# Magn Synthesizer 🎹

**Magn** is a high-fidelity, web-based musical instrument designed for sound exploration and performance. Built with a modern industrial aesthetic, it provides a powerful set of synthesis tools directly in your browser.

## 🚀 Features

- **Multi-Shape Oscillator**: Choose between Sine, Sawtooth, Square, Triangle, and Pulse waveforms.
- **Advanced Visualizers**: Real-time visual feedback with four display modes:
  - **Waveform**: See the raw output of your sound.
  - **Spectroscope**: Analyze the frequency distribution.
  - **Spectrogram**: A scrolling heatmap of your spectral content.
  - **Lissajous**: Beautiful XY plots of the audio signal.
- **ADSR Envelope**: Precise control over the sound's evolution (Attack, Decay, Sustain, Release).
- **Resonant Filter**: Sculpt your tone with adjustable Cutoff and Resonance.
- **LFO Modulation**: Add movement to your sound with controllable Rate and Amplitude.
- **Interactive Keyboard**: A responsive 25-key interface for immediate playability.

## 🛠️ Tech Stack

- **Core**: TypeScript & Web Audio API
- **Build Tool**: Vite
- **Styling**: Vanilla CSS with a focus on industrial design and premium aesthetics.

## 🏁 Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run locally**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## 🌐 Navigation

The interface includes a **Home** button that connects you back to the [Phantom Studio](https://phantomstudio-five.vercel.app/) ecosystem.
